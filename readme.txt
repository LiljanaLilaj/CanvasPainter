How to run:

Unzip and open the zip files in IntelliJ as project and run the main class: CanvasExercise

or 

Use CanvasPainter.jar to run from cmd line (file under CanvasPainter dir in the zip)
1. Change dir to where CanvasPainter.jar is
1. Run command: java -jar CanvasPainter.jar

How te enter the commands:

Please enter a command from below:

1. C w h  - Create a new canvas of width w and height h
2. L x1 y1 x2 y2  - Draw a new line from coordinates  to horizontally or vertically. Lines are made up of the  characte (x1, y1) (x2, y2) x r
3. R x1 y1 x2 y2 - Draw a new rectangle, with upper left corner at coordinate  and lower right coordinate at . Lines are (x1, y1) (x2, y2) made up of the  character x
4. Q - Quit the program


Start with 1. to first create a Canvas and then use the rest to draw a shape or quit.
Letters are not case sensitive.
